How do I use this app?

It's almost self explanatory, but here we go. To track different coins you can click on the button on each card. You can have a maximum of five coins selected. If you select more, a modal window will prompt you to re-think your choices. You can also go to the Live Reports tab to track the coins you selected!

So how did I make this?

You can look at my github for more detailed information (Linked below). Basically, I worked really hard - studied a lot - and tried to teach myself different ways of doing the same thing. For example - If I found a way to store something in local storage, I would try to find another way so that I could expand my skill set.
